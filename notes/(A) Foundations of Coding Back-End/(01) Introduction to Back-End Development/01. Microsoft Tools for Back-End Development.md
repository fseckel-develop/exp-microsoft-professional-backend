## Overview  

- Program uses Microsoft ecosystem tools for modern back-end development  
- Core stack includes:  
	- C#  
	- .NET / ASP.NET Core  
	- GitHub  
	- GitHub Copilot  
	- Microsoft Copilot  
	- Visual Studio Code (VS Code)  
- Focus: Performance, collaboration, AI-assisted productivity, scalable cloud-ready solutions  

---
## Why Use C# for Back-End Development  

- **Performance & Scalability**  
	- Optimised through the .NET runtime  
	- Suitable for high-performance, enterprise-grade systems  

- **.NET Integration**  
	- Tight integration with ASP.NET Core  
	- Simplifies building secure and robust APIs  

- **Cross-Platform Support**  
	- .NET Core enables development on Windows, macOS, and Linux  
	- Ideal for cloud-native and containerised deployments  

- **Strong Typing & Object-Oriented Design**  
	- Encourages structured, maintainable architecture  
	- Reduces runtime errors through compile-time checks  

- **Large Ecosystem**  
	- Extensive libraries, tools, and frameworks  
	- Strong community and enterprise backing

## Introduction to GitHub  

- Platform for version control and collaboration  
- Built on Git for tracking code changes and managing history  

### Key Features  

- **Version Control**  
	- Tracks every modification  
	- Allows rollback to previous versions  
	- Supports multiple contributors  

- **Collaboration**  
	- Pull requests for structured code review  
	- Issues for tracking bugs and tasks  
	- Facilitates real-time teamwork  

- **Integration**  
	- Works seamlessly with VS Code  
	- Connects with cloud platforms for deployment workflows  

- **Open Source Ecosystem**  
	- Hosts millions of projects  
	- Enables reuse and contribution to shared libraries  

- Essential for maintaining organised and professional development workflows

## GitHub Copilot vs. Microsoft Copilot  

### GitHub Copilot  

- Embedded directly in VS Code  
- Provides real-time, context-aware code suggestions  
- Assists with:  
	- Writing C# code  
	- Understanding logic  
	- Inline debugging support  
- Improves coding speed and problem-solving skills  

### Microsoft Copilot  

- Integrated with Microsoft 365  
- Focused on:  
	- Code refinement  
	- Summarisation  
	- Explanation and documentation  
- Helps identify potential issues  
- Enhances clarity and maintainability of solutions

## Introduction to Visual Studio Code (VS Code)  

- Lightweight and flexible code editor from Microsoft  
- Ideal for back-end development and rapid iteration  
- Supports multiple languages including C#, HTML, and JavaScript  

### Why We Use VS Code  

- **Customization**  
	- Large extension marketplace  
	- Tailor the editor for specific workflows  

- **Cross-Platform Support**  
	- Runs on Windows, macOS, and Linux  

- **Version Control Integration**  
	- Built-in Git support  
	- Seamless GitHub connectivity  

- Combines simplicity with powerful development features for efficient back-end engineering
