## Overview

- Logical processes are the fundamental building blocks of programming
- They define the flow and control within code, ensuring programs operate correctly
- Key logical processes: **sequences**, **control structures**, and **iterations**

---
## Sequences

- The simplest control structure: instructions are executed in a specific, linear order
- Real-life analogy: following a morning routine (brush teeth → shower → breakfast)
- Ensures program steps occur in the correct order to achieve desired results
- Example (pseudocode):
    - Step 1: Open applications folder
    - Step 2: Open text editor
    - Step 3: Create a new document

---
## Control Structures (Conditional Statements)

- Determines the flow of code based on specific conditions
- Enables programs to respond differently depending on inputs or states
- Real-life analogy: deciding to go for a walk if it’s not raining
- Common form: **if-else statements** (alternative form: **switch statements**)
- Example (pseudocode):
    - If the age field is blank → prompt the user
    - Else → move to the next field

---
## Iterations (Loops)

- Allows repeated execution of a block of code until a specific condition is met
- Real-life analogy: practicing a skill repeatedly until mastery
- Useful for processing multiple items or performing repetitive tasks
- Example (pseudocode for database processing):
    - Load a record
    - Capitalise the last name field
    - If last record → exit
    - Else → move to next record and repeat

---
## Importance of Logical Processes

- Computers operate on logic, so understanding logical structures is essential for programming
- Helps break down complex problems into manageable parts
- Essential for writing **efficient, error-free, and maintainable code**

---
## Practical Example: Task List App

- **Sequences**: Add tasks in order
- **Control structures**: Mark tasks as complete/incomplete using if-then logic
- **Iterations**: Save and load tasks from memory/storage repeatedly for all tasks
- Logical processes ensure smooth operation and a better user experience

---
## Summary

- Logical processes form the foundation of programming
- **Sequences**, **control structures**, and **iterations** guide program flow
- Applying these structures enables developers to write structured, efficient, and reliable code
