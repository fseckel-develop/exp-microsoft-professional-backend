## Overview

- Logical operations in programming use operators such as **AND**, **OR**, and **NOT** to manipulate **true/false values** (the truth value of a boolean expression)
- These operators are essential for **conditional logic** and **program flow control**
- Key logic types: **Boolean logic**, **conditional statements**, **loops**.

---
## Boolean Logic

- Deals with **true/false values** (truth values)
- Evaluates expressions to guide program decisions using boolean operators

- **AND**-Operator: True only if both expressions are true
	- Example: Both first name and last name fields have values → AND = true

- **OR**-Operator: True if at least one expression is true
	- Example: Either first name or last name has a value → OR = true

- **NOT**-Operator: Negates the truth value of a single expression
	- Example: Last name is not null → NOT(null) = true

---
## Conditional Statements

- Use Boolean values to control the program flow

- **If statements**: Execute a block of code if a condition is true.
    - Example: Prompt user if first name or last name is null.

- **Switch statements**: Handle multiple possible values of a variable.
    - Example: Display country-specific license agreement based on the country field.
    - Includes a **default case** for unmatched values.

---
## Loops (Iterations)

- Allow repeated execution of code blocks

- **For loop**: Repeats a block a **specific number of times**
    - Example: Process 10 database records to remove spaces from names.

- **While loop**: Repeats a block **until a condition is false**.
    - Example: Delete duplicate records from a database until none remain.

---
## Summary

- **Boolean logic** evaluates expressions and determines truth values
- **Conditional statements** (if, switch) guide program flow based on conditions
- **Loops** (for, while) handle repetitive tasks dynamically
- Together, these logical structures allow programmers to create **dynamic, efficient, and responsive programs**
