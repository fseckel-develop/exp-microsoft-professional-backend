
- **Algorithms are step-by-step instructions** used to solve problems or perform tasks
- They guide program flow, decision-making, and data processing
- Mastering structures like **conditional**, **categorical**, and **binary** logic is essential for writing clear and efficient code

---
## Conditional Statements

- Allow programs to **make decisions** based on true/false conditions
- Used to choose different execution paths

- **If/Then Statements:**
    - Execute actions when a condition is true
    - Example:
        - _If age ≥ 18 → print “Eligible to vote”_
        - _Else → print “Not eligible to vote”_

- **Switch Statements:**
    - Efficiently handle **multiple possible cases**.
    - Example: weekly planner choosing tasks based on the day.

---
## Categorical Statements

- Used to **organise and classify data** based on criteria
- Helpful for grouping datasets logically

- **Example (Attendee Age Groups):**
    - _If age < 13 → Children_
    - _Else if 13–19 → Teens_
    - _Else → Adults_

- Creates structured outputs for easier analysis and decision-making

---
## Binary Structures

- Decisions with **two possible outcomes** (yes/no, true/false)
- Fundamental for fast logical evaluation

- **Example (Age-Restricted Area):**
    - _If age ≥ 21 → add to “21+”_
    - _Else → add to “under 21”_

- Produces exactly two groups for simplified processing

---
## Conclusion

- Understanding **conditional**, **categorical**, and **binary** structures enables developers to create **organised, logical, and efficient algorithms**
- These structures form the foundation for solving complex programming tasks and improving code clarity
