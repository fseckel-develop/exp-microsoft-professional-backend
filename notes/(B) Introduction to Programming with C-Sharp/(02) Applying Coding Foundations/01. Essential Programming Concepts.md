## Boolean Logic and Control Structures

- Boolean logic is used for decision-making in programming
- Three primary operations:
	- **AND** returns true only if all conditions are true  
	- **OR** returns true if at least one condition is true
	- **NOT** inverts a Boolean value, turning true into false and vice versa

- Control structures use Boolean logic to set the program flow:
	- **if-else statements** execute code blocks based on conditions
		  Example: Open a door in a game if the player has the key
	- **else statements** provide alternative actions when conditions aren’t met
	- **switch statements** handle multiple potential cases efficiently
- allow programs to make decisions dynamically and respond to varying inputs

---
## Using Loops for Repetition

- Loops allow repetitive execution of code until a condition changes:
	- **For loops** repeat a block of code a set number of times
		  Example: Iterating through items in a list
	- **While loops** continue executing as long as a condition remains true
		  Example: Keep attacking enemies in a game until none remain
- Loops automate repetitive tasks, reduce manual coding, and help prevent errors.

## Organising Code with Methods

- Methods are reusable blocks of code that perform specific tasks
- improve program organisation and maintainability
- Structure of a method: **name**, **parameters (inputs)**, **return type (output)**, and **code**
- Example in C#:

```csharp

{

    return velocity;

````

- Benefits of methods:
	- Break down complex problems into smaller, manageable parts
	- Improve readability, maintainability, and testability

## Planning with Pseudocode

- Pseudocode outlines program logic in plain language without worrying about syntax
	- Helps map algorithm steps before coding
	- Ensures logical flow and avoids early errors

- Example: Car acceleration function pseudocode:
	1. Initialise Car
	2. Check Gas Level
	3. Increase Velocity if Gas > 0
	4. Display Updated Status

- Pseudocode is a key planning tool to clarify structure and guide development
