## Newtonsoft.Json

A widely used library for processing JSON data in .NET applications. Key Features:

- **JSON parsing**
    - Converts JSON from APIs into C# objects.
    - Example: Turning e-commerce order history JSON into usable C# models.

- **Serialisation**
    - Converts C# objects into JSON strings.
    - Example: Sending order details to a shipping service.

- **Deserialisation**
    - Converts JSON responses into C# objects.
    - Example: Converting tracking info (status, delivery date) into usable objects.

## Dapper

A lightweight micro-ORM for efficient database interactions. Key features:

- **Data querying & mapping**
    - Maps SQL query results directly to C# objects.
    - Example: Fetching e-commerce order records quickly and easily.

- **High performance**
    - Much faster than full ORMs; ideal for high-traffic systems.

- **Simplicity**
    - Minimal overhead, great for small apps with basic data access needs.

---
## Serilog

A structured logging library ideal for monitoring .NET applications. Key features:

- **Error tracking**
    - Captures detailed logs including contextual data (e.g., order IDs on failure).

- **User activity monitoring**
    - Tracks behaviours such as product page visits or cart abandonment.

- **Log routing**
    - Sends logs to multiple destinations (file, cloud, central log server).
