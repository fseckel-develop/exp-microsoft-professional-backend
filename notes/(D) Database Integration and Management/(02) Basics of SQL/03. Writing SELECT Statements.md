
- SQL is a powerful tool for managing and analysing data. 
- The **SELECT statement** is the foundation of retrieving data from databases. 

---
### Using SELECT Statements for Data Retrieval

- **Purpose:** Retrieve specific columns or all data from a table.

- **Syntax for specific columns:**

```sql
SELECT column1, column2 FROM table_name;
```

- **Syntax to retrieve all columns:**

```sql
SELECT * FROM table_name;
```

#### Example: 

Retrieve first and last names of employees from the Employees table:

```sql
SELECT firstName, lastName FROM Employees;
```


---
### Removing Duplicate Values with DISTINCT

- **Purpose:** Ensure query results contain only unique values.

- **Syntax:**

```sql
SELECT DISTINCT column_name FROM table_name;
```

#### Example: 

Retrieve unique department names:

```sql
SELECT DISTINCT department FROM Employees;
```


---
### Limiting Results with TOP

- **Purpose:** Retrieve only the top N rows based on a condition.

- **Syntax:**

```sql
SELECT TOP N column_name(s) FROM table_name 
ORDER BY column_name DESC;
```

#### Example: 

Retrieve the top 5 highest-paid employees:

```sql
SELECT TOP 5 * FROM Employees ORDER BY Salary DESC;
```


---
### Filtering Data with WHERE

- **Purpose:** Retrieve rows that meet specific conditions.

- **Syntax:**

```sql
SELECT column_name(s) FROM table_name 
WHERE condition;
```

- **Comparison operators:** =, <, >, <=, >=, <>
- **Logical operators:** AND, OR

#### Examples:

- Employees in the HR department:

```sql 
SELECT firstName, lastName FROM Employees 
WHERE Department = ‘HR’;
```

- Employees in HR with salaries greater than $50,000:

```sql
SELECT firstName, lastName FROM Employees 
WHERE Department = ‘HR’ AND Salary > 50000;
```

- Employees in HR or Finance with more than 5 years of experience:

```sql
SELECT firstName, lastName, Department FROM Employees 
WHERE (Department = ‘HR’ OR Department = ‘Finance’) AND YearsExperience > 5; 
```

---
### Sorting Data with ORDER BY

- **Purpose:** Organise query results in ascending or descending order.

- **Syntax:**

```sql
SELECT column_name(s) FROM table_name 
ORDER BY column_name ASC | DESC;
```

- **Options:** ASC = ascending order (default), DESC = descending order

#### Examples:

- Sort employees by last name alphabetically:

```sql 
SELECT firstName, lastName FROM Employees 
ORDER BY lastName ASC;
```

- Sort HR employees by salary, highest first:

```sql
SELECT firstName, lastName, Salary FROM Employees 
WHERE Department = ‘HR’ ORDER BY Salary DESC;
```


---
### Summary

- **SELECT** retrieves data from a database.
- **WHERE** filters data based on conditions.
- **ORDER BY** sorts data in ascending or descending order.
- **DISTINCT** removes duplicates.
- **TOP** limits the number of returned rows.

Mastering these commands allows precise data retrieval, enabling insights and 
better decision-making from large datasets.
