## Basic SQL Functions

#### CONCAT:

Joins multiple text strings into one. Use case: Combine first and last names into a full name.

```sql
SELECT 
	CONCAT(FirstName, ’ ’, LastName) AS FullName 
	FROM MusicCatalog;
```

#### LEN / LENGTH:

Counts the number of characters in a text field, including spaces and punctuation.

```sql
SELECT 
	LEN(LastName) AS LastNameLength 
	FROM MusicCatalog;
```

#### UPPER and LOWER:

Converts text to uppercase or lowercase for standardisation.

```sql
SELECT 
	UPPER(FirstName) AS FirstNameUpper, 
	LOWER(Email) AS EmailLower 
	FROM MusicCatalog;
```

#### SUBSTRING:

Extracts a portion of text from a string based on starting position and length.

```sql
SELECT 
	SUBSTRING(LastName, 1, 3) AS LastNameSnippet 
	FROM MusicCatalog;
```

## Aggregate Functions

#### COUNT:

Returns the number of rows meeting a condition.

```sql
SELECT COUNT( * ) AS TotalSongs FROM MusicCatalog;
```

#### SUM:

Adds all values in a column.

```sql
SELECT SUM(Revenue) AS TotalRevenue FROM MusicCatalog;
```

#### AVG:

Calculates the average of values in a column.

```sql
SELECT AVG(Revenue) AS AverageRevenue FROM MusicCatalog;
```

#### MIN:

Finds the smallest value in a column.

```sql
SELECT MIN(Revenue) AS LowestRevenue FROM MusicCatalog;
```

#### MAX:

Finds the largest value in a column.

```sql
SELECT MAX(WeeklyPlays) AS HighestPlays FROM MusicCatalog;
```

#### GROUP BY:

Groups rows with the same values in a column, used with aggregate functions.

```sql
SELECT Genre, SUM(Revenue) AS TotalRevenue FROM MusicCatalog GROUP BY Genre;
```

---
## Summary

- **Basic Functions:** 
  Manipulate and transform text data (CONCAT, LEN, UPPER, LOWER, SUBSTRING).
- **Aggregate Functions:** 
  Summarise data for insights (COUNT, SUM, AVG, MIN, MAX).
- **GROUP BY:** 
  Enhances aggregate functions by grouping data for meaningful metrics.
