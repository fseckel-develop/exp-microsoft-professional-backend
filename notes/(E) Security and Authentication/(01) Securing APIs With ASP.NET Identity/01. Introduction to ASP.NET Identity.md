## Overview of ASP.NET & ASP.NET Identity

- ASP.NET is a Microsoft web development framework for building dynamic, scalable, and secure web applications
- Primarily uses **C#** for development
- ASP.NET Identity is a **membership system** within ASP.NET
	- Manages user authentication, authorisation, and roles
	- Ensures secure access control within web applications

---
## Core Identity Concepts

### Authentication (AuthN)

- Verifies a user's identity before granting access
- Commonly done via username and password
- Ensures only legitimate users can access protected resources
- Enhanced security features include:
	- Multi-Factor Authentication (MFA)
	- Email confirmation during registration
	- Password recovery via email links

### Authorisation (AuthZ)

- Determines what an authenticated user is allowed to do
- Controls access based on permissions or roles
- Prevents unauthorised actions within the application
- Example:
	- Employees can view personal data
	- Managers can approve team requests

### User Roles

- Roles group permissions instead of assigning them individually
- Simplifies permission management
- Users inherit permissions based on assigned roles
- Example roles:
	- Admin: full system access
	- Customer: browse and purchase products
	- Merchant: manage inventory

---
## Key ASP.NET Identity Components

- **User Manager**:
	- Handles user account operations
	- Responsibilities include:
		- Creating, updating, and deleting users
		- Managing passwords and user claims
	- Enables account lifecycle management

- **Sign-In Manager**:
	- Manages user authentication processes
	- Handles sign-in and sign-out logic
	- Validates user credentials securely

- **Role Manager**:
	- Defines and manages roles within the application
	- Assigns users to specific roles
	- Enforces role-based access control

- **IdentityDbContext**:
	- Connects ASP.NET Identity to the database
	- Securely stores:
		- User credentials
		- Roles
		- Claims and permissions
	- Ensures persistence of identity data across sessions

## Integration & Architecture

#### ASP.NET Core Integration

- ASP.NET Core provides the foundation for ASP.NET Identity
- Enables:
	- Cross-platform application development
	- Modular and customisable identity management
- Supports external authentication providers:
	- Google
	- Facebook
	- OAuth 2.0-based logins

#### Data Storage with Entity Framework

- Uses **Entity Framework (EF)** for database interactions
- Stores user data securely in relational databases
- Benefits:
	- Object-based data access using C#
	- No need for complex SQL queries
	- Simplified data management and retrieval
