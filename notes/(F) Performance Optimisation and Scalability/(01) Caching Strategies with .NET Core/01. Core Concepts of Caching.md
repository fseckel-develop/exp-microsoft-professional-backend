## Definition and Purpose of Caching

- Caching is the temporary storage of frequently accessed data
- Data is stored in faster-to-access locations such as:
	- RAM
	- Browser cache
- Reduces the need to repeatedly fetch or recompute data
- Commonly used to improve speed, efficiency, and user experience in applications

---
## Performance Optimisation Through Caching

- Improves application speed and responsiveness
- Performance optimisation focuses on:
	- Reducing response times
	- Lowering resource usage
	- Enhancing overall user experience
- Cached data loads faster than data retrieved from original sources
	- Example: cached images or CSS files load instantly

---
## Key Benefits of Caching

- Response Time Reduction
	- Eliminates repeated data retrieval
	- Enables instant loading for repeat visits
- Reduced Database Queries
	- Prevents excessive database access
	- Improves application performance during peak traffic
- Optimised Server Resources
	- Reduces redundant data processing
	- Helps servers handle traffic spikes efficiently
- Latency Management
	- Stores data closer to users
	- Minimises delay between request and response

---
## Scalability and High-Traffic Handling

- Supports consistent performance during heavy usage
- Reduces server load by serving cached content
- Examples:
	- Online sales using cached product lists
	- News sites caching popular articles during breaking news
	- Travel sites caching frequent search results

---
## Common Use Cases for Caching

- Web Page Components
	- Images
	- Navigation menus
	- Stylesheets
- Frequently Accessed API Endpoints
	- Prevents redundant processing
	- Reduces network latency
- Session Data
	- Maintains user state across visits
	- Keeps shopping carts active for returning users
- Product and Content Pages
	- Enables smooth browsing during peak traffic

---
## Conclusion

- Caching is a foundational technique for building fast, scalable applications
- Minimises delays and optimises system resources
- Ensures reliable performance under real-world usage demands
- Plays a critical role in delivering responsive, user-friendly experiences
