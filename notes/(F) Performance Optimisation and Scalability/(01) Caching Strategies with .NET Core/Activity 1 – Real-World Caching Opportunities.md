## Overview

- Learners explore real-world applications to identify cacheable data
- Focus on understanding:
	- Performance optimisation
	- Scalability
	- User experience improvements
- Applications analysed:
	- Online store
	- Video streaming site
	- Messaging app

---
## Example 1: Online Store

### Cacheable Data

- Product catalog
	- Images
	- Descriptions
	- Prices
- Search results
- User session data
	- Shopping cart

### Benefits of Caching

- Faster load times
	- Product images and details load instantly
- Reduced server load
	- Fewer repeated database queries
- Improved scalability
	- Supports more concurrent users without slowdown

---
## Example 2: Video Streaming Site

### Cacheable Data

- Video thumbnails and metadata
	- Titles
	- Descriptions
- Recently watched videos list
- Popular video recommendations

### Benefits of Caching

- Reduced latency
	- Thumbnails display instantly
- Efficient bandwidth usage
	- Fewer backend and API calls
- Smoother user experience
	- Faster access to frequently viewed content

## Example 3: Messaging App

### Benefits of Caching

- Instant message retrieval

	- Reduced repeated message fetching
- Seamless user experience
	- Contacts and assets load immediately
