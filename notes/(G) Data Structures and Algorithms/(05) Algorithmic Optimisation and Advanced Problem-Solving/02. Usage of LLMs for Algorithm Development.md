## Introduction

- Large language models (LLMs), such as **Microsoft Copilot**, enhance back-end development by automating code generation, refactoring, and optimisation.  
- These tools help developers create efficient, scalable systems faster while focusing on refining algorithmic logic.  
- LLMs are particularly useful for implementing complex algorithms like **Dijkstra's shortest path** and **A\*** search, which optimise data flow and reduce computation time in real-time applications.  

---
## Key Benefits of LLMs in Code Development

- **Automated Code Creation**  
	- Generate foundational code structures using natural language prompts  
	- Reduces setup and boilerplate time  

- **Refactoring Assistance**  
	- Identifies inefficient patterns  
	- Suggests cleaner, optimised alternatives  

- **Performance Optimisation**  
	- Provides suggestions to improve execution speed and memory usage  
	- Enhances scalability and maintainability of large applications  

---
## Enhancing Algorithmic Development

### Dijkstra’s Algorithm

- Finds shortest path in weighted graph, accounting for connection costs (time, distance, etc.) 
- Applications: network routing, mapping, optimising data transfer  
- Benefits: ensures efficient data flow, reduces computation time, minimises delays  

### A* Search Algorithm

- Builds on Dijkstra’s approach using a heuristic to estimate the best path  
- Applications: GPS navigation, video game AI, API request prioritisation  
- Benefits: prioritises promising paths, improves response times, and enhances scalability  

### Role of Microsoft Copilot

- Assists in **implementation** of complex algorithms in .NET Core  
- Provides **real-time analysis** of time and space complexity  
- Suggests **refactorings** to improve algorithm performance  
- Ensures algorithms are optimised for high-performance back-end systems
