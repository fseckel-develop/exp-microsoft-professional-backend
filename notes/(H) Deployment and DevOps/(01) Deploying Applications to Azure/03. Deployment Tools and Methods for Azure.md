## Core Azure Deployment Tools

### Azure App Service

- Fully managed Platform-as-a-Service (PaaS)  
- Used for building, deploying, and scaling web apps and APIs  
- Azure manages infrastructure and scaling  
	- Developers focus on application logic  
- Supports:  
	- Automatic scaling  
	- High availability  
- Ideal for hosting production-grade applications in the cloud  

### Azure CLI (Command Line Interface)

- Command-line tool for managing Azure resources  
- Enables quick creation, configuration, and management of services  
- Reduces reliance on graphical web interfaces  
- Supports automation and scripting  
- Useful for:  
	- Creating databases  
	- Configuring servers  
	- Managing deployments  

### GitHub Actions

- Automation platform integrated into GitHub  
- Enables Continuous Integration and Continuous Deployment (CI/CD)  
- Automates:  
	- Building  
	- Testing  
	- Deploying applications  
- Integrates directly with Azure services  
	- Deploy to Azure App Service  
	- Deploy to Azure Functions  
- Minimises manual deployment effort  

### Integrated Terminal in VS Code

- Built-in terminal within Visual Studio Code  
- Allows execution of commands without leaving the editor  
- Supports:  
	- Resource management  
	- Application testing  
	- Deployment to Azure  
- Streamlines development and deployment within a single workspace  

---
## Deployment Workflow Overview

- Develop application code  
- Use Integrated Terminal or Azure CLI for local testing and resource management  
- Automate build and deployment with GitHub Actions  
- Deploy and host using Azure App Service  
- Achieve scalable, high-availability production environments
