
For a logistics system (Inventory + Orders), roles should reflect operational responsibility:

### (1) Admin

- Full system control
- Can:
	- Manage users & roles
	- Create/update/delete inventory
	- Create/update/delete orders
	- Adjust order items
	- View everything
- This is your superuser.

### (2) WarehouseStaff

- Inventory-focused role
- Can:
	- View inventory
	- Create/update inventory
	- Adjust stock
	- View orders (read-only)
- Cannot:
	- Delete orders
	- Manage users

### (3) SalesStaff

- Order-focused role
- Can:
	- Create orders
	- Add/remove order items
	- Adjust quantities
	- View inventory (read-only)
- Cannot:
	- Modify inventory prices
	- Delete inventory items
	- Manage users

### (4) Viewer (Optional)

- Read-only role for:
	- Auditors
	- Managers
	- Reporting dashboards
- Can:
	- View inventory
	- View orders
- Cannot:
	- Create / Update / Delete anything

---
# Role Model Summary

| Role           | Inventory | Orders | Users |
| -------------- | --------- | ------ | ----- |
| Admin          | Full      | Full   | Full  |
| WarehouseStaff | CRUD      | Read   | None  |
| SalesStaff     | Read      | CRUD   | None  |
| Viewer         | Read      | Read   | None  |
