# (A61) StudentGradeManager

A console-based Student Grade Management System that demonstrates basic object-oriented design, collection usage, and menu-driven interaction in C#.

---
## Course Context

**Course**: Foundations of Coding Back-End  
**Section**: Project Planning and Design

This project implements a simple system based on defined functional and non-functional requirements and demonstrates how a design outline can be translated into code.

---
## Concepts Demonstrated

- Basic object-oriented design
- Collections (`List`, `Dictionary`)
- Input validation
- Menu-driven console applications
- Separation of concerns (Models / Services / Presentation)

---
## Project Structure

- **Models** – Domain model representing students and their grades
- **Services** – Application logic for managing students and grades
- **Presentation** – Console menu and formatted output
- **Program.cs** – Application entry point and dependency setup
