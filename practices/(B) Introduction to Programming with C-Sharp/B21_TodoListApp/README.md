# (B21) ToDoListApp

A console-based to-do list application that demonstrates logical program flow, basic object-oriented design, and task management using a menu-driven interface in C#.

---
## Course Context

**Course**: Introduction to Programming with C#  
**Section**: Essential Programming Concepts

This project applies fundamental programming concepts such as Boolean logic, control structures, loops, and methods to build a small interactive application. It also demonstrates logical thinking techniques like problem decomposition and structured debugging during development.

---
## Concepts Demonstrated

- Boolean logic and control structures (`if`, `switch`)
- Loops for program flow (`while`)
- Method-based program organisation
- Basic object-oriented design
- Logical problem decomposition
- Input validation and debugging practices
- Separation of concerns (Models / Services / Presentation)

---
## Project Structure

- **Models** – Task domain model and task status definitions  
- **Services** – Application logic for adding, listing, and completing tasks  
- **Presentation** – Console menu and formatted output  
- **Program.cs** – Application entry point and application setup
