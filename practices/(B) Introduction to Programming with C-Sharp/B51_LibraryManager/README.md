# (B51) LibraryManager

A console-based library management application that demonstrates structured C# development through book management, borrowing workflows, and clean separation between domain logic and user interaction.

---
## Course Context

**Course**: Introduction to Programming with C#  
**Section**: AI-Assisted Development with GitHub Copilot

This project demonstrates how AI-assisted coding can support the development of a small but structured application. It applies generated and refactored code within a layered console project while keeping the implementation readable, maintainable, and consistent.

---
## Concepts Demonstrated

- AI-assisted code generation and refactoring
- Basic object-oriented design
- Menu-driven console applications
- Input validation and error handling
- State management for borrowing and returning books
- Modular project organisation
- Separation of concerns (Models / Services / Presentation)

---
## Project Structure

- **Models** – Book domain model and borrowing state  
- **Services** – Application logic for managing books and library actions  
- **Presentation** – Console menu and formatted output  
- **Program.cs** – Application entry point and application setup
