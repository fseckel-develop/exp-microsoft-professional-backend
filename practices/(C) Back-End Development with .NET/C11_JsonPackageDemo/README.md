# (C11) JsonPackageDemo

A console application demonstrating JSON serialisation and deserialisation in C# using the built-in `System.Text.Json` library.

---
## Course Context

**Course**: Back-End Development with .NET  
**Section**: Package Management with NuGet

This project demonstrates how external libraries can be integrated into a .NET application to handle structured data formats. It focuses on converting C# objects to JSON and reconstructing objects from JSON strings, illustrating common data exchange patterns used in APIs and backend systems.

---
## Concepts Demonstrated

- Using NuGet packages and external libraries
- JSON serialisation and deserialisation
- Data transfer object modelling
- Handling structured data formats
- Basic service-based architecture
- Separation of concerns (Models / Services / Presentation)

---
## Project Structure

- **Models** – Book data model used for JSON conversion  
- **Services** – JSON serialisation and deserialisation logic  
- **Presentation** – Console output formatting for JSON and book data  
- **Program.cs** – Demonstrates JSON conversion and object reconstruction
