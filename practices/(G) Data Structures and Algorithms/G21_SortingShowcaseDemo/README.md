# (G21) SortingShowcaseDemo

A console application demonstrating and benchmarking **Bubble Sort, Quick Sort, and Merge Sort** on a large product dataset to compare their behaviour, performance, and scalability.

---
## Course Context

**Course**: Data Structures and Algorithms  
**Section**: Sorting Algorithms

This project showcases how different sorting algorithms perform under the same scenario. It compares simple quadratic-time sorting with more scalable divide-and-conquer approaches, highlighting why algorithm choice matters for real-world backend systems.

---
## Concepts Demonstrated

- Bubble Sort
- Quick Sort
- Merge Sort
- Algorithm benchmarking and comparison
- Time complexity trade-offs
- Stability and memory usage considerations
- Sorting large in-memory datasets

---
## Project Structure

- **Algorithms** – Sorting algorithm implementations and shared sorter interface  
- **Benchmarking** – Benchmark runner, scenarios, and performance reports  
- **Data** – Product model and random dataset generation  
- **Presentation** – Console output for benchmark results  
- **Program.cs** – Application entry point and benchmark orchestration
