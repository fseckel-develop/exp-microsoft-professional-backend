# (G22) SearchingShowcaseDemo

A console application demonstrating and benchmarking **Linear Search and Binary Search** on large employee datasets to compare their speed, requirements, and scalability.

---
## Course Context

**Course**: Data Structures and Algorithms  
**Section**: Searching Algorithms

This project demonstrates how different search strategies behave depending on dataset structure. It compares linear search on unsorted data with binary search on sorted data, showing why algorithm choice has a major impact on backend performance.

---
## Concepts Demonstrated

- Linear Search
- Binary Search
- Benchmarking search performance
- Searching sorted vs unsorted datasets
- Time complexity trade-offs
- Large dataset lookup scenarios
- Search algorithm selection based on data structure

---
## Project Structure

- **Algorithms** – Search algorithm implementations and shared search interface  
- **Benchmarking** – Benchmark runner, scenarios, and search reports  
- **Data** – Employee record model and dataset generation utilities  
- **Presentation** – Console output for benchmark results  
- **Program.cs** – Application entry point and benchmark orchestration
