# (G31) TreeStructuresDemo

A console application demonstrating **tree data structures and traversal strategies** through a content catalog example. The project compares general trees, binary trees, binary search trees, and AVL trees to show how hierarchy, ordering, and balancing affect navigation and lookup performance.

---
## Course Context

**Course**: Data Structures and Algorithms  
**Section**: Tree Structures and Traversal

This project demonstrates how tree-based structures organise hierarchical and searchable data more efficiently than linear scanning. It also shows how traversal strategies and self-balancing trees improve structured navigation and lookup performance in backend-style scenarios.

---
## Concepts Demonstrated

- General trees for hierarchical category structures
- Binary trees for two-branch hierarchical organization
- Binary search trees for ordered lookup
- AVL trees for self-balancing search performance
- Tree traversal methods (`PreOrder`, `InOrder`, `PostOrder`, `LevelOrder`)
- Comparing lookup depth between BST and AVL trees
- Modeling hierarchical data in a domain-driven example

---
## Project Structure

- **Models** – Domain models for content items, categories, datasets, and search results  
- **Trees** – Tree implementations and node types  
- **Scenarios** – Demonstrations comparing tree types, traversals, and lookup behavior  
- **Presentation** – Console output and traversal formatting helpers  
- **Program.cs** – Application entry point and scenario runner
