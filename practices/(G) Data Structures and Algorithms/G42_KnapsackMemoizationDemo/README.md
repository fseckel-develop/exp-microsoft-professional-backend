# (G42) KnapsackMemoizationDemo

A console application that compares a brute-force recursive solution and a memoized dynamic programming solution to the 0/1 Knapsack problem.

---
## Course Context

**Course**: Data Structures and Algorithms  
**Section**: Dynamic Programming and Memoization

This project demonstrates how dynamic programming improves performance by caching intermediate results. It compares a naive recursive approach with a memoized solver to highlight how avoiding repeated subproblems significantly reduces computation.

---
## Concepts Demonstrated

- 0/1 Knapsack optimisation problem
- Recursive brute-force algorithms
- Memoization (top-down dynamic programming)
- Subproblem caching
- Performance benchmarking
- Resource allocation modelling

---
## Project Structure

- **Algorithms** – Recursive and memoized knapsack solvers
- **Benchmarking** – Execution measurement and solver comparison
- **Data** – Sample dataset representing expedition supplies
- **Models** – Domain models for items and solver results
- **Presentation** – Console output formatting
- **Program.cs** – Application entry point
