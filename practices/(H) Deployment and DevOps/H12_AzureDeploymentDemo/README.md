# (H12) AzureDeploymentDemo

A Blazor WebAssembly application demonstrating how to provision Azure infrastructure and deploy a .NET web application to Azure App Service using the Azure CLI and a scripted deployment workflow.

---
## Course Context

**Course**: Cloud Deployment and DevOps Foundations  
**Section**: Azure Deployment and Cloud Infrastructure

This project demonstrates the end-to-end workflow of deploying a web application to Azure. It covers resource provisioning, App Service setup, and automated deployment using the Azure CLI, illustrating how local applications transition into cloud-hosted services.

---
## Concepts Demonstrated

- Azure App Service hosting
- Azure CLI resource management
- Cloud resource provisioning (resource groups, App Service plans)
- Deployment automation with scripts
- Blazor WebAssembly cloud deployment
- Azure deployment workflow
- Infrastructure setup using command-line tools

---
## Project Structure

- **scripts** – PowerShell script for provisioning Azure resources and deploying the application
- **src** – Blazor WebAssembly application source code
